QR Code Reader Website

This is a simple and lightweight QR Code Reader web application built using HTML, CSS, and JavaScript. It uses the html5-qrcode library to access the device camera and scan QR codes directly in the browser.

🚀 Features

📷 Real-time camera QR scanning

⚡ Fast and lightweight

💻 Works in modern browsers

📱 Mobile-friendly

🎯 Displays scanned results instantly

🛠 Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

html5-qrcode

📦 How It Works

The browser requests camera access.

The QR scanner detects and decodes QR codes.

The decoded content is displayed instantly on the webpage.

🔧 How to Run

Clone the repository.

Open index.html in your browser.

Allow camera permissions.

Start scanning QR codes.
